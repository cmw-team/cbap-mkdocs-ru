Local Git hooks configuration

To enable repository-local hooks in this project, run:

```
git config core.hooksPath .githooks
```

Hooks included:
- prepare-commit-msg: Suggest `[#XXXXX]` prefix on stderr from feature-branch name or recent commits (does not mutate the message)
- commit-msg: Warn if message does not start with `[#XXXXX]` (does not block)
